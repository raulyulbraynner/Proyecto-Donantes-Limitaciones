# Proyecto-Donantes-Limitaciones
El proyecto +Donantes -Limitaciones permitirá realizar una aplicación móvil donde los usuarios de una forma rápida y segura podrán interactuar y realizar su donación, de igual manera permitirá que la fundación “Una mano por los discapacitados” pueda llevar un control y seguimiento de las donaciones realizadas a los usuarios beneficiarios.
